Bei der Erkundung können Sie gelegentlich verlassene Schiffe oder Basen entdecken. Diese können wichtige Informationen enthalten, die Sie erhalten können, wenn Sie an Bord gehen und sie untersuchen. Sie können in der Regel auch geborgen werden, indem Sie ein Konstruktions-Schiff schicken, um sie zu reparieren.

[H3]Liste der verlassenen Schiffe und Basen[/H3]
Dieser Bildschirm listet alle bekannten verlassenen Schiffe und Basen auf, auf die Sie bei Ihrer Erkundung der Galaxie gestoßen sind.

Wenn Sie auf die Galaxie- oder Systemebene herauszoomen, werden die Standorte der verlassenen Schiffe und Basen auf der Karte hervorgehoben. Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird der Standort auf der Karte angezeigt. Wenn Sie auf das Objekt klicken, wird das verlassene Schiff oder der Stützpunkt ausgewählt. Ein Doppelklick auf das Objekt verschiebt die Ansicht zu diesem Ort.

Mit der Schaltfläche auf der rechten Seite können Sie die Reparatur und Untersuchung des verlassenen Schiffes oder der Basis in die Warteschlange stellen. Dadurch wird ein Konstruktions-Schiff geschickt, um das Objekt zu reparieren.
