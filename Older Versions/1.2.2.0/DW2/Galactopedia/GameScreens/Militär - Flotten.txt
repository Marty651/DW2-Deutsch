Flotten sind Gruppierungen von Militärschiffen, die gemeinsam agieren, um Missionen auszuführen.

Sie haben eine Heimatbasis, die ihr Operationszentrum definiert. Sie haben auch eine Einsatzreichweite, die den Bereich definiert, in dem sie feindliche Ziele angreifen, in einem Radius von ihrer Heimatbasis aus.

[H3]Flottenliste[/H3]
Dieser Bildschirm listet alle Flotten in Ihrem Reich auf. Probieren Sie die Sortier- und Filteroptionen am oberen Rand des Fensters.

Wenn Sie auf die Galaxie- oder Systemebene herauszoomen, werden die Flotten auf der Karte hervorgehoben. Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird das Gebiet auf der Karte angezeigt. Wenn Sie auf das Objekt klicken, wird es ausgewählt. Ein Doppelklick auf das Objekt verschiebt die Ansicht auf dieses Gebiet.

Jedes Objekt in der Liste zeigt die aktuelle Mission der Flotte an. Außerdem werden die Anzahl der Schiffe in der Flotte, die Kampfstärke der Flotte und die Angriffsstärke der Truppen in der Flotte angezeigt. Das Symbol für die Flottenrolle wird ganz rechts angezeigt. Alle Charaktere in der Flotte werden rechts neben der Missionsbeschreibung angezeigt.

Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird eine detaillierte Zusammenfassung in einem Fenster auf der rechten Seite angezeigt. Dieses Fenster zeigt eine vollständige Aufschlüsselung aller Schiffe der Flotte nach Schiffsrolle, Schadensstärke und Treibstoffvorrat. Sie enthält auch Details zu allen Admirälen und Generälen in der Flotte.

[H3]Flottenvorlagen und Flottenrollen[/H3]
Wie bereits erläutert, basieren Flotten auf einer Vorlage. Flotten können eine bestimmte Rolle haben, wie z.B.: Angreifen, Überfallen, Verteidigen oder Erobern. Diese Flotten wählen die Missionen aus, die zu ihrer Flottenrolle passen. 

Sie können eine Flotte auch manuell steuern, indem Sie ihre Rolle auf Manuell setzen (Schaltfläche Flottenrolle im Auswahlfenster) oder indem Sie der Flotte einfach eine Mission zuweisen (wodurch sie automatisch zu einer manuellen Flotte wird).

[H3]Neue Flotten erstellen[/H3]
Sie können eine neue Flotte auf verschiedene Arten erstellen:
[ULI]Wählen Sie mehrere Militärschiffe in der Hauptansicht aus, indem Sie sie mit gedrückter linker Maustaste auswählen oder indem Sie auf jedes Schiff mit der Strg-Taste klicken. Klicken Sie dann auf die Schaltfläche 'Neue Flotte erstellen' unter dem Auswahlfenster[/ULI]
[ULI]Wählen Sie mehrere Schiffe in der Liste der Militärschiffe aus, indem Sie bei gedrückter Strg- und Shift-Taste auf die Schiffe in der Liste klicken. Klicken Sie dann auf die Schaltfläche 'Neue Flotte erstellen' unter dem Auswahlfenster[/ULI]
[ULI]Wählen Sie im Bildschirm Flottenvorlagen die gewünschte Vorlage aus und klicken Sie dann entweder auf die Schaltflächen 'Neue Flotte bilden' oder 'Neue Flotte bauen' auf der rechten Seite[/ULI]

Sie können auch einzelne Schiffe zu einer bestehenden Flotte hinzufügen, indem Sie das Schiff auswählen und auf die Schaltfläche 'Schiff zur Flotte hinzufügen' im Auswahlfenster klicken. Daraufhin wird eine Liste mit allen Flotten angezeigt, denen Sie beitreten können. Wählen Sie die Flotte aus, der das Schiff beitreten soll, indem Sie sie in der Liste anklicken.

[H3]Taktische Einstellungen[/H3]
Flotten verfügen über eine ganze Reihe von taktischen Einstellungen, mit denen Sie ihr Vorgehen feinabstimmen können. Öffnen Sie den Bildschirm mit den taktischen Einstellungen, indem Sie die Flotte auswählen und auf die Schaltfläche 'Taktik einstellen' ganz rechts unter dem Auswahlfenster klicken.

Sie können steuern, wie die Flotten auf Bedrohungen reagieren, indem Sie die Einsatzreichweite einstellen. Weitere Details erhalten Sie, wenn Sie den Mauszeiger über die einzelnen Einstellungen im Bildschirm Taktische Einstellungen bewegen.

Beachten Sie, dass auch die einzelnen Schiffe innerhalb einer Flotte ihre eigenen taktischen Einstellungen haben. Wie Flotten- und Schiffstaktik zusammenspielen, können Sie im Abschnitt 'Flottenschiffe - Taktische Überlagerungen' einstellen.
