Bauschiffe sind mobile Bauwerften. Sie bauen Basen verschiedener Art: Bergbaustationen, Forschungsstationen, Erholungsbasen, Überwachungsstationen. Sie können auch unbewegliche beschädigte Schiffe reparieren. Sie können sogar verlassene Schiffe und Basen reparieren, auf die Sie bei der Erkundung gestoßen sind.

Für den Bau eines neuen Stützpunktes werden Baumaterialien benötigt. Die erste Aktion eines Bauschiffs, das einen Bauauftrag erhält, besteht also darin, zum nächstgelegenen Raumhafen zu reisen, um die benötigten Baumaterialien zu laden.

[H3]Liste der Bauschiffe[/H3]
Dieser Bildschirm listet alle Bauschiffe in Ihrem Reich auf.

Wenn Sie auf die Galaxie- oder Systemebene herauszoomen, werden die Bauschiffe auf der Karte hervorgehoben. Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird die Position auf der Karte angezeigt. Wenn Sie auf den Gegenstand klicken, wird er ausgewählt. Ein Doppelklick auf das Objekt verschiebt die Ansicht an diesen Ort.

Jedes Listenelement zeigt die aktuelle Mission des Bauschiffs an.

[H3]Automatisierungsstatus[/H3]
Mit der kleinen Schaltfläche auf der rechten Seite können Sie den Automatisierungsstatus des Schiffes ändern. So können Sie das Bauschiff manuell steuern oder es vollständig automatisieren, so dass es seine eigenen Missionen auswählen kann.

Oder Sie können eine spezifischere Automatisierung vornehmen, bei der das Bauschiff nur Bergbaustationen, Forschungsstationen, Überwachungsstationen, Erholungsbasen baut oder verlassene Schiffe und Basen birgt.
