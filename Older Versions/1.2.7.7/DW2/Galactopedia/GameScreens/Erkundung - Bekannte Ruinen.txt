Bei der Erkundung können Sie an manchen Orten antike Ruinen entdecken. Diese Ruinen können Geheimnisse und andere Objekte von Wert enthalten.

[H3]Liste der bekannten Ruinen[/H3]
Dieser Bildschirm listet alle Ruinen auf, auf die Sie bei Ihrer Erkundung der Galaxie gestoßen sind.

Wenn Sie auf die Galaxie- oder Systemebene herausgezoomt haben, werden die Ruinenstandorte auf der Karte hervorgehoben. Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird der Standort auf der Karte angezeigt. Wenn Sie auf das Objekt klicken, wird sein Standort ausgewählt. Ein Doppelklick auf das Objekt verschiebt die Ansicht zu diesem Ort.

Jedes Objekt in der Liste zeigt alle relevanten Boni an, die die Ruinen bieten. Sie können diese Boni für Ihr Reich erlangen, indem Sie den Planeten, auf dem sich die Ruinen befinden, kolonisieren. Sie können Forschungsboni erhalten, indem Sie eine Forschungsstation an diesem Ort errichten. Sie können Landschaftsboni erhalten, indem Sie an diesem Ort eine Erholungsbasis errichten.

Wenn Sie den Mauszeiger über eine Ruine in der Liste bewegen, wird auf der rechten Seite eine detaillierte Übersicht angezeigt.
