Migration bedeutet im Allgemeinen, dass Menschen aus großen Kolonien in neue, kleinere Kolonien abwandern. Die Migration trägt dazu bei, die Bevölkerung neuer Kolonien schnell zu vergrößern, so dass diese schneller produktiv werden können.

Die Migranten werden mit privaten Passagierschiffen zu ihrem Ziel transportiert. Migration findet ausschließlich im privaten Sektor Ihres Reiches statt - Sie haben keine direkte Kontrolle über die Bewegung von Bürgern zwischen Ihren Kolonien.

[H3]Migrationsfaktoren[/H3]
Zu den Faktoren, die neue Migranten in eine Kolonie locken, gehören:
[ULI]niedrige Steuersätze[/ULI]
[ULI]allgemeine Attraktivität: hohe Planetenqualität, hohes Entwicklungsniveau[/ULI]
[ULI]wie nah die neue Kolonie an der derzeitigen Heimatkolonie des Migranten liegt[/ULI]

Zu den Faktoren, die die Auswanderung von einer Kolonie weg fördern, gehören:
[ULI]Zufriedenheit in der Kolonie: Unzufriedenheit fördert die Auswanderung[/ULI]
[ULI]hohe Bevölkerungszahl: stark bevölkerte Kolonien fördern die Auswanderung[/ULI]
[ULI]nahegelegene Kolonien für Auswanderung erwünscht[/ULI]

[H3]Migration zwischen den Reichen[/H3]
Es ist wahrscheinlicher, dass Menschen innerhalb ihres eigenen Reiches auswandern, einfach weil es einfacher ist - es gibt weniger Bürokratie zu überwinden.

Wenn Sie jedoch einen Migrationsvertrag mit einem anderen Reich abgeschlossen haben, kann es auch zu einer Migration zwischen den Reichen kommen.

Sobald Ihr Reich eine Milliarde Bürger einer bestimmten außerirdischen Rasse enthält, erhält Ihr Reich auch deren Rassenboni. Dazu gehören z.B. intelligentere Wissenschaftler und Ingenieure, bessere Spione, schnellere Minenarbeiter, usw.

[H3]Bevölkerungspolitik der Kolonie[/H3]
Mit der Bevölkerungspolitik können Sie festlegen, welche fremden Rassen Sie in Ihren Kolonien zulassen.

Diese können im Bevölkerungspolitik-Bildschirm festgelegt werden, der sich auf der Auswahlleiste öffnet, wenn eine deiner Kolonien ausgewählt wird. Bewegen Sie den Mauszeiger über das Symbol für die Bevölkerungspolitik, das sich ganz rechts in der Bevölkerungsleiste der ausgewählten Kolonie befindet. Klicken Sie auf das Symbol oder auf den Bildschirm, um ihn zu öffnen.

Jede Rasse, die Sie kennengelernt haben, wird aufgelistet, so dass Sie für jede fremde Rasse in jeder Kolonie spezifische Richtlinien festlegen können.

Mit diesen Einstellungen haben Sie eine genaue Kontrolle über die Migration in Ihre Kolonien.
