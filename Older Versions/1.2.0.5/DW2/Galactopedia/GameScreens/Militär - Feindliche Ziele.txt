Dieser Bildschirm listet bekannte feindliche Ziele auf, die Sie mit Ihren Flotten angreifen können. Dazu gehören feste Ziele wie feindliche Basen (z.B. Piratenbasen) und Kolonien. Sie enthält auch feindliche Flotten, deren aktuelles Gebiet wir kennen.

Wenn Sie auf die Galaxie- oder Systemebene herauszoomen, werden die feindlichen Ziele auf der Karte hervorgehoben. Wenn Sie den Mauszeiger über ein Objekt in der Liste bewegen, wird das Gebiet auf der Karte angezeigt. Wenn Sie auf das Objekt klicken, wird sein Gebiet ausgewählt. Ein Doppelklick auf das Objekt verschiebt die Ansicht auf dieses Gebiet.

Jedes Objekt in der Liste zeigt die bekannte Stärke des Feindes auf dem Gebiet des Ziels an. Bei feindlichen Basen wird auch die aktuelle Gesundheitsstufe des Ziels angezeigt.

[H3]Verteidigungsstärke[/H3]
Für feindliche Kolonien wird hier die bekannte Stärke der verteidigenden Truppen angezeigt. Beachten Sie, dass es sich hierbei um eine Schätzung handeln kann, wenn die Kolonie nicht direkt sichtbar ist, so dass Vorsicht geboten ist, wenn Sie eine Invasion erwägen.

[H3]Angriffe in die Warteschlange stellen[/H3]
Mit der Schaltfläche auf der rechten Seite können Sie einen Angriff gegen das Ziel in die Warteschlange stellen. Wenn die nächste geeignete Flotte verfügbar ist, wird sie einen Angriffsauftrag gegen das Ziel erteilen. Sie können die Schaltfläche erneut anklicken, um den Angriff abzubrechen.

Wenn Sie das Ziel nicht angreifen können, ist die Schaltfläche Warteschlange deaktiviert. Ein Tooltip auf der Schaltfläche erklärt, warum Sie nicht angreifen können.
